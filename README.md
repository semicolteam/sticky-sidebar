# Sticky Sidebar
A lightweight jquery plugin for make any side bar sticky!

# How to use?

Include jquery.stickysidebar.min.js in your website header:

```ruby
<script src="jquery.stickysidebar.min.js"></script>
<script>
  $('#side-bar-id').stickysidebar();
</script>
```
# Option

```ruby
  $('#side-bar-id').stickysidebar({
    offset: 50
  });
```
